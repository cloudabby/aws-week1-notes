# 🟢 Day 1 — What is Cloud Computing?

## ☁️ What is Cloud Computing?

Cloud computing is the delivery of computing services (servers, storage, databases, networking, software) over the internet instead of using physical hardware.

It allows businesses to rent infrastructure instead of buying servers.

---

## 🖥 What is a Server?

A server is a powerful computer that:
- Stores data
- Hosts websites
- Runs applications
- Responds to client requests

Example:
When I open a website, my browser sends a request to a server, and the server responds with website data.

---

## 💻 Local Computer vs Cloud

Local Computer:
- Fixed hardware
- Limited scalability
- High upfront cost

Cloud:
- Virtual infrastructure
- Scalable on demand
- Pay-as-you-go pricing

---

## ☁️ What AWS Provides

AWS provides:
- Compute (EC2)
- Storage (S3)
- Databases (RDS)
- Networking (VPC)
- Security (IAM)

AWS removes the need to manage physical data centers.
