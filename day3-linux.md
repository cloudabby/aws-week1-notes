# 🟢 Day 3 — Linux & Operating Systems

## 🖥 What is an Operating System?

An OS manages:
- Hardware
- Memory
- Processes
- Files

Examples:
- Windows
- Linux
- macOS

---

## 🐧 What is Linux?

Linux is an open-source operating system widely used in cloud computing.

Most AWS servers run Linux.

---

## 💻 Linux Commands

| Command | Description |
|----------|--------------|
| ls | Lists files in directory |
| cd | Changes directory |
| pwd | Shows current directory |
| mkdir | Creates new folder |
| rm | Removes file or folder |
