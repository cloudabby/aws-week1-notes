# 🟢 Day 2 — How the Internet Works

## 🌐 What is an IP Address?

An IP address is a unique number assigned to a device on a network.
Example: 192.168.1.1

It allows devices to communicate over the internet.

---

## 🌍 What is DNS?

DNS (Domain Name System) converts domain names into IP addresses.
Example:
google.com → 142.x.x.x

Without DNS, we would need to memorize IP addresses.

---

## 🌐 What is HTTP?

HTTP is the protocol used for communication between client and server.

HTTPS = Secure version of HTTP.

---

## 🖥 Client vs Server

Client:
- Sends request (browser)

Server:
- Responds with data

---

## 🔎 DevTools Observations

Status Codes:
- 200 → Success
- 301 → Redirect
- 404 → Not Found
- 500 → Server Error
