# 🟢 Day 4 — Networking Fundamentals

## 🌐 What is Networking?

Networking connects devices to share data.

---

## 🔢 What is a Subnet?

A subnet divides a large network into smaller networks.

Used for:
- Organization
- Security
- Traffic management

---

## 🚪 What is a Gateway?

A gateway connects one network to another.

Usually connects private network to the internet.

---

## 🔄 What is NAT?

NAT (Network Address Translation) allows private IP addresses to access the internet using a public IP.
